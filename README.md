# Momentum

A modern, private and offline-first productivity dashboard that runs entirely in the browser.

## Features

- Task manager with priority filters
- Pomodoro-style focus timer
- Generated brown noise for concentration
- Daily habit tracker with streaks
- Hydration tracker
- Autosaving quick notes
- 60-second guided breathing reset
- Dark and light themes
- Installable PWA with offline support
- Local-only storage; no account, analytics or backend

## Run locally

Open `index.html`, or serve the folder with any static server:

```bash
python -m http.server 8000
```

## GitHub Pages

The included workflow deploys the repository as a static GitHub Pages site whenever `main` changes.

Built with plain HTML, CSS and JavaScript.
